# Website Quality Scorecard Reference

Complete scoring criteria for AI/B2B startup website assessment.

## I. Messaging & Clarity (Max: 18 points)

Measures how effectively the site communicates purpose and audience in 5-20 seconds.

### A. Specificity in Headline (0-3)
Is the headline highly specific using user's technical jargon?

| Score | Example |
|-------|---------|
| 0 | "The Future of Work" |
| 1 | "Better Productivity Tools" |
| 2 | "Project Management for Teams" |
| 3 | "Issue Tracking for Engineering Teams" / "DCF Models for Analysts" |

### B. User-Centric Language (0-3)
Uses terminology the ideal technical/enterprise user resonates with?

| Score | Indicator |
|-------|-----------|
| 0 | Founder jargon, invented terms |
| 1 | Generic business speak |
| 2 | Some user terminology |
| 3 | Speaks exactly like target users |

Avoid: "Progressive discovery", internal concept names

### C. Clear Audience Definition (0-3)
Immediately clear *who* this is for?

| Score | Indicator |
|-------|-----------|
| 0 | Could be for anyone |
| 1 | Vague professional category |
| 2 | Specific role mentioned |
| 3 | "For [specific role] who [specific problem]" |

Examples: engineers, financial analysts, college football programs

### D. Literal Sub-Headline (0-3)
Marketing headline supported by literal product description?

| Score | Indicator |
|-------|-----------|
| 0 | No sub-headline or equally vague |
| 1 | Somewhat clarifying |
| 2 | Describes product function |
| 3 | Crystal clear what the product does |

### E. Focused Messaging (0-3)
Homepage concise? Acts as front page guiding deeper?

| Score | Indicator |
|-------|-----------|
| 0 | Overwhelming feature dump |
| 1 | Too many messages competing |
| 2 | Mostly focused with some clutter |
| 3 | Clean, guides users to explore more |

### F. Naming Simplicity (0-3)
Straightforward naming without confusion?

| Score | Indicator |
|-------|-----------|
| 0 | Multiple confusing names (company + product + feature names) |
| 1 | Some naming confusion |
| 2 | Minor complexity |
| 3 | Single clear name throughout |

## II. Design & Aesthetic Quality (Max: 15 points)

Visual execution, polish, professionalism, and credibility signals.

### A. Avoidance of "AI Slop" (0-3)
Free from generic AI-generated aesthetics?

| Score | Indicator |
|-------|-----------|
| 0 | Obvious AI/template look (purple gradients, massive shadows, poor typography) |
| 1 | Multiple AI-slop indicators |
| 2 | Mostly original with minor issues |
| 3 | Distinctly crafted, original aesthetic |

Red flags: Excessive gradients, overly smooth/soulless feel, generic iconography

### B. Visual Consistency (0-3)
Consistent styling across all pages?

| Score | Indicator |
|-------|-----------|
| 0 | Major inconsistencies |
| 1 | Noticeable style variations |
| 2 | Mostly consistent |
| 3 | Perfect consistency (fonts, colors, components) |

Check: Marketing pages, product screenshots, pricing, roadmap

### C. Strategic Visuals (0-3)
Screenshots/visuals used effectively?

| Score | Indicator |
|-------|-----------|
| 0 | Busy, unfamiliar UI dumps |
| 1 | Generic or confusing visuals |
| 2 | Adequate but not compelling |
| 3 | Build mystery OR clearly demonstrate key workflows |

### D. Contrast and Legibility (0-3)
Sufficient contrast for all elements?

| Score | Indicator |
|-------|-----------|
| 0 | Critical legibility issues |
| 1 | Multiple contrast problems |
| 2 | Minor issues |
| 3 | Excellent legibility throughout |

Watch: Yellow text on light backgrounds, low-contrast CTAs

### E. Professionalism / Trust Signal (0-3)
Aesthetic signals trustworthiness for target audience?

| Score | Indicator |
|-------|-----------|
| 0 | Unprofessional, amateur feel |
| 1 | Inconsistent quality |
| 2 | Adequate professionalism |
| 3 | Mature, enterprise-ready aesthetic |

Critical for enterprise buyers who "check boxes"

## III. User Experience & Conversion Flow (Max: 18 points)

Clarity of user path and effectiveness of interactive elements.

### A. Clear Primary CTA (0-3)
Single focused CTA per screen/scroll?

| Score | Indicator |
|-------|-----------|
| 0 | No clear CTA or many competing |
| 1 | Multiple CTAs fighting for attention |
| 2 | Primary CTA identifiable |
| 3 | One obvious next step per screen |

### B. Minimal Distractions (0-3)
Hero section free from distracting animations?

| Score | Indicator |
|-------|-----------|
| 0 | Major distracting motion/animation |
| 1 | Noticeable distractions |
| 2 | Minor movements |
| 3 | Focus stays on core message |

### C. Conversational UI Integrity (0-3)
Interactive elements properly placed and styled?

| Score | Indicator |
|-------|-----------|
| 0 | Chat/prompts look like cookie banners |
| 1 | Confusing placement |
| 2 | Adequate but not optimal |
| 3 | Interactive elements clearly intentional |

### D. Accessible Demo/Trial (0-3)
Easy way to try before enterprise demo/payment?

| Score | Indicator |
|-------|-----------|
| 0 | Demo requires enterprise contact only |
| 1 | Hidden or difficult access |
| 2 | Available but not prominent |
| 3 | Self-serve trial/demo easily accessible |

### E. Showcasing Workflows (0-3)
Complex products visually demonstrate *how* they work?

| Score | Indicator |
|-------|-----------|
| 0 | Just lists capabilities |
| 1 | Minimal workflow visualization |
| 2 | Some workflow demonstration |
| 3 | Clear connected tools/flowchart examples |

For integration/workflow products especially

### F. Timely Monetization (0-3)
Avoids scaring users with early pricing?

| Score | Indicator |
|-------|-----------|
| 0 | Pricing/payment upfront before value |
| 1 | Pricing prominent too early |
| 2 | Balanced presentation |
| 3 | Value-first, pricing appropriately timed |

---

## IV. Credibility & Positioning (Max: 12 points)

Company stage, backing, and expertise presentation.

### A. Authentic Stage Presentation (0-3)
Stage-appropriate image? Owns being a startup?

| Score | Indicator |
|-------|-----------|
| 0 | Pretending to be large enterprise |
| 1 | Mixed signals about stage |
| 2 | Mostly authentic |
| 3 | Confidently owns current stage |

False enterprise image sets wrong expectations

### B. Credibility Hierarchy (0-3)
Social proof secondary to product description?

| Score | Indicator |
|-------|-----------|
| 0 | Leads with investors/logos over product |
| 1 | Unbalanced emphasis on backing |
| 2 | Appropriate balance |
| 3 | Product first, proof supports |

Logos, investors, GitHub stars support—not lead

### C. Accurate Proof (0-3)
All credibility claims current and factual?

| Score | Indicator |
|-------|-----------|
| 0 | Outdated or inaccurate claims |
| 1 | Some questionable claims |
| 2 | Mostly accurate |
| 3 | All claims verifiable and current |

### D. Focus on Expertise (0-3)
Founder/team expertise in problem domain visible?

| Score | Indicator |
|-------|-----------|
| 0 | No expertise signals |
| 1 | Generic team page |
| 2 | Some domain expertise shown |
| 3 | Clear unique insight/expertise displayed |

---

## Score Interpretation

| Range | Grade | Interpretation |
|-------|-------|----------------|
| 51-60 | A | Excellent—ready for scale |
| 41-50 | B | Good—minor refinements needed |
| 31-40 | C | Adequate—significant improvements possible |
| 21-30 | D | Problematic—major issues to address |
| 0-20  | F | Critical—fundamental rework needed |
